# NLW-2-PROFFY
 
 Projeto desenvolvido durante a 2ª Next Level Week, com o professor da RockeatSeat, Mayk Brito.

Plataforma que tem como objetivo auxilia a conexão entre professores e alunos

Tecnologias usadas:
---------------------------
  - HTML
  - CSS
  - JavaScript
  - [NodeJS][1]
  - [Nodemon][2]
  - [Nunjucks][3]
